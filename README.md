# kaggle-dev-env

Docker environment for kaggle

## 要件

* kaggle notebook の実行環境と同様の環境で開発ができる
* ssh によるリモート開発が可能
* github 自動連携
* コマンドにより kaggle のデータ取得、学習、notebook の提出を自動化できる
* mlflow による実験管理ができる
* dvc により実験データ管理ができる
